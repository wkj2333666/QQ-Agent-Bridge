# 联网搜索

用于百度、全网搜索、公开资料、新闻、价格、人物经历、政策、版本、时间敏感事实。

## Rules

- 必须先用 WebSearch、浏览器或可用网页工具，再写结论。
- 不要凭模型记忆回答实时信息；无法联网时写“无法联网搜索：<原因>”。
- 关键结论要给来源 URL；没有可靠来源就写“未找到可靠来源”。
- 多来源冲突时说明冲突，不要合并成确定事实。
- 只报告实际成功的搜索步骤；不要写“查了多个来源”除非真的查到。

## Output

- 简短结论优先；需要整理时用“事项 / 时间 / 来源 URL / 可信度 / 备注”。
- 人物经历、公司、政策、新闻等必须区分公开资料、推测和未验证信息。
- 搜索完成判定：至少一次工具调用；关键事实有来源 URL。
