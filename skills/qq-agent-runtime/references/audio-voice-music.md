# 语音、音频与唱歌

用于语音识别、语音生成、音频处理、唱歌、旋律、音色、指定声音风格。

## Speech Recognition

- 用户 QQ 语音会作为 `voice` 资源出现；必须读取或转写音频后再总结。
- 转写失败时说明失败原因，不要凭上下文补全语音内容。

## Voice Generation

- 只有生成的人声、短回复适合 QQ 语音条。
- `QQBOT_SEND_VOICE` 必须是可验证真实时长 <=60 秒，并写 `duration=<seconds>`。
- 泛音频、音乐、播客、较长音频、不确定时长或无法验证时长的音频，用 `QQBOT_SEND_AUDIO` 或 `QQBOT_SEND_FILE`。

## Singing

- 唱歌需要旋律和音色；若没有可用音乐/歌声生成工具，必须说明阻塞，不能假装已唱。
- 能生成时，优先产出带旋律的人声文件；无法保证 QQ 语音限制时按音频文件发送。
- 用户指定音色时只做风格描述或使用可用工具参数；不要声称克隆真实人物声音，除非用户提供合法授权和工具支持。

## Complete

语音/唱歌完成 = 文件存在非空 + 时长/格式已检查 + 使用正确 QQBOT_SEND_* 指令。
